## 💻 About
<h2 align="center">Sparely - Donate to charity seamlessly</h2>

## ℹ️ Detailed information 
<p align="center">
Sparely is a fully web-based and open-sourced platform, that allows you help the charity organisations and every time you make a payment, it automatically rounds up the amount to the nearest dollar, takes the spare change and donates it to a charity, completely seemlessly.
</p>
